## WannaCry Ransomware Introduction 
WannaCry / WannaCrypt simply means that it encrypts the files on infected Windows systems. There are two key components – a worm and a ransomware package
it spreads laterally between computers on the same LAN by using a
vulnerability in implementations of Server Message Block (SMB) in Windows
systems. It also spreads through malicious email attachments.
